Configurer un Mac 100 % privé en 2 heures (guide 2026)

macOS est globalement meilleur que Windows côté vie privée par défaut, mais loin d'être parfait. Apple collecte beaucoup de télémétrie « pour améliorer ses produits », et de nombreux réglages critiques sont désactivés par défaut. Voilà le protocole pour blinder ton Mac sans rien casser de tes usages quotidiens.

Ce guide marche pour macOS Sequoia, Sonoma et Ventura (macOS 13+). Si tu es sur une version plus ancienne, mets-la à jour d'abord — c'est la première règle de sécurité.

Phase 1 — Gestionnaire de mots de passe (15 min)

Premier réflexe avant tout le reste. iCloud Keychain (le gestionnaire intégré Apple) est correct mais limité à l'écosystème Apple. Si tu utilises aussi un PC ou Android, ou si tu veux du contrôle total, passe à un vrai gestionnaire multiplateforme.

  1. Installe Bitwarden depuis le Mac App Store ou bitwarden.com/download
  2. Installe l'extension Safari/Brave/Firefox
  3. Crée ton compte avec un mot de passe maître long (25+ caractères, mémorisable)
  4. Active la 2FA sur ton compte Bitwarden (authenticator app + code de secours imprimé)
  5. Importe tes mots de passe iCloud Keychain : Préférences Système → Mots de passe → ⋯ → Exporter → CSV → Import dans Bitwarden

Pour le tutoriel détaillé, voir notre guide Bitwarden complet.

Phase 2 — FileVault (chiffrement disque complet, 10 min)

Sans FileVault activé, si quelqu'un te vole ton Mac, il peut booter depuis une clé USB Linux et lire tous tes fichiers en clair. Avec FileVault, c'est mort.

Sur les Mac récents (Apple Silicon M1/M2/M3/M4), FileVault est souvent activé par défaut depuis macOS 12. Mais vérifie quand même :

  1. Menu Pomme → Réglages Système → Confidentialité et sécurité → FileVault
  2. Si tu vois "FileVault est activé pour le disque…" → ✅ tu es protégé
  3. Sinon, clique Activer
    • macOS te demande où sauvegarder la clé de récupération : compte iCloud (pratique mais Apple a une copie) ou clé locale (à imprimer et ranger en sûreté)
    • Pour la paranoia maximale → clé locale + impression papier rangée dans un coffre
  4. Le chiffrement initial prend quelques heures en arrière-plan (utilisation normale possible pendant ce temps)

⚠️ La clé de récupération est CRITIQUE. Si tu la perds ET tu oublies ton mot de passe macOS, tes données sont perdues à jamais. Pas de récupération possible — c'est exactement ce qui fait que FileVault est sûr.

Phase 3 — Lockdown Mode (mode paranoia, optionnel)

Apple a introduit le Mode Lockdown en 2022 pour les cibles à haut risque (journalistes, activistes, personnalités). Il désactive automatiquement :

  • La plupart des pièces jointes (sauf images)
  • Les liens preview dans Messages
  • Les profils de configuration installables
  • Les connexions par câble quand le Mac est verrouillé
  • FaceTime et appels de numéros inconnus

Pour activer (si tu en as besoin) :

Réglages Système → Confidentialité et sécurité → Mode isolement (Lockdown Mode) → Activer

⚠️ Ce mode casse beaucoup de fonctionnalités du quotidien (certains sites web ne marchent plus, certains messages ne s'ouvrent pas). À activer SEULEMENT si tu es une cible spécifique. Pour 99 % des gens, c'est de la sur-protection.

Phase 4 — VPN avec kill switch (15 min)

Sur Mac, le VPN doit :

  • Démarrer avec ton Mac
  • Se connecter automatiquement
  • Couper tout trafic si la connexion VPN tombe (kill switch / Lockdown mode)

Avec Mullvad sur macOS :

  • Téléchargement depuis mullvad.net ou via Homebrew : brew install mullvad-vpn
  • Settings → Launch on start-up : ON
  • Auto-connect : ON
  • Lockdown mode (kill switch) : ON
  • Protocol : WireGuard
  • DNS content blockers : Ads + Trackers + Malware ON

Test final : mullvad.net/check → trois feux verts.

Si tu veux plus de pays disponibles et un débridage Netflix US / Disney+ / BBC iPlayer, NordVPN est le meilleur compromis. App native macOS, Threat Protection anti-malware DNS intégrée, plans dès ~3 €/mois en 2 ans.

Phase 5 — Réglages de confidentialité macOS (20 min)

C'est la partie la plus longue mais la plus importante. Pomme → Réglages Système → Confidentialité et sécurité → on parcourt chaque section.

Localisation

Préférences → Confidentialité et sécurité → Service de localisation. Désactive pour toutes les apps qui n'en ont pas besoin (mais garde pour Maps, Météo, Localiser si tu les utilises).

Analyse et amélioration

Désactive TOUT :

  • Partager les données d'analyse Mac avec Apple → OFF
  • Améliorer Siri et Dictée → OFF
  • Partager avec les développeurs d'apps → OFF
  • Partager avec les iCloud Analytics → OFF

Publicité Apple

Réglages → Confidentialité et sécurité → Publicité Apple :

  • Publicités personnalisées → OFF

Accessibilité, Micro, Caméra, Photos, Fichiers

Passe en revue chaque app autorisée. Désactive l'accès pour celles qui n'en ont pas besoin. Tu peux laisser pour Zoom, WhatsApp, Discord, Photos système, etc.

Spotlight Suggestions

Réglages → Spotlight → décoche :

  • Suggestions Siri
  • Suggestions Safari
  • Apps web

Ça empêche Spotlight d'envoyer tes recherches à Apple/Bing.

Apple ID et iCloud

Réglages → ton nom (en haut) → iCloud :

  • Active la protection avancée des données (Advanced Data Protection) → chiffrement de bout en bout pour TOUS tes fichiers iCloud (par défaut, Apple peut lire certains de tes fichiers iCloud)
  • Si tu actives cette option, sauvegarde ta clé de récupération iCloud impérativement (imprimée + rangée en sûreté)

Phase 6 — Pare-feu (Firewall)

macOS a un pare-feu intégré mais désactivé par défaut. À activer :

  1. Réglages → Réseau → Pare-feu
  2. Active Pare-feu
  3. Clique Options → coche "Activer le mode furtif" (rend ton Mac invisible aux scans réseau)

Pour aller plus loin avec un pare-feu applicatif (contrôle quelles apps peuvent communiquer) : Little Snitch (~45 €, mais c'est l'outil de référence Mac) ou LuLu (gratuit, open source, de Patrick Wardle aka Objective-See).

Phase 7 — Le navigateur (15 min)

Safari sur Mac est relativement bon pour la vie privée (Intelligent Tracking Prevention activé par défaut, sans cookies tiers). Mais si tu veux le maximum :

Option A — Safari durci (le plus intégré)

  • Safari → Réglages → Confidentialité
  • Cocher "Bloquer tous les cookies" (peut casser certains sites)
  • "Masquer mon adresse IP des traceurs" → ON
  • Désactiver "Suggestions Safari"

Option B — Brave (recommandé)

  • Téléchargement : brave.com
  • Bloqueur de pubs et trackers natif et agressif
  • Mode Private avec Tor intégré
  • Fingerprinting protection forte

Que tu choisisses Safari ou Brave : désactive Chrome. Chrome est l'outil de tracking n°1 de Google.

Phase 8 — Désactiver le démarrage automatique des apps

Beaucoup d'apps installent des "Login Items" / "Background Items" qui démarrent avec ton Mac sans te le dire.

  1. Réglages → Général → Ouverture/Fermeture de session
  2. Onglet "Au démarrage de la session" : supprime tout ce qui n'est pas absolument nécessaire
  3. Onglet "Autoriser en arrière-plan" : désactive les apps qui n'ont pas besoin de tourner H24

Garde activé : Mullvad VPN, Bitwarden, et c'est tout. Désactive le reste sauf si vraiment indispensable.

Phase 9 — Chiffrement de disques externes (VeraCrypt)

Pour chiffrer une clé USB ou un disque externe transporté avec le Mac :

  • VeraCrypt est compatible Mac → voir notre guide complet VeraCrypt
  • Alternative native macOS : Utilitaire de disque → Effacer → format "APFS (chiffré)" → mot de passe. Plus simple mais ne marche que pour les lecteurs lus depuis un Mac.

Phase 10 — Sauvegarde chiffrée (Time Machine)

Si tu utilises Time Machine, active impérativement le chiffrement de la sauvegarde :

  1. Réglages → Général → Time Machine
  2. Sélectionne ton disque → coche "Chiffrer la sauvegarde"
  3. Définis un mot de passe (différent de ton mot de passe macOS, stocké dans Bitwarden)

Sans ça, n'importe qui qui vole ton disque Time Machine récupère toutes tes données.

Vérification finale

  1. mullvad.net/check → 3 feux verts
  2. browserleaks.com/webrtc → aucune IP locale visible
  3. haveibeenpwned.com → vérifie si tes emails apparaissent dans des fuites
  4. Dans Préférences Système → Confidentialité et sécurité → FileVault → doit être ACTIVÉ
  5. Dans Réseau → Pare-feu → doit être ACTIVÉ

Différences Mac vs Windows — récap

Aspect Mac (macOS) Windows
Chiffrement disque FileVault (souvent activé par défaut sur Apple Silicon) BitLocker (Pro) ou VeraCrypt (Home)
Télémétrie par défaut Modérée (Apple Analytics) Forte (Microsoft Telemetry)
Pare-feu Désactivé par défaut, à activer Windows Defender Firewall activé
Anti-malware XProtect intégré (silencieux, efficace) Windows Defender (visible, efficace)
App Store sandbox Forte (les apps Mac App Store sont isolées) Variable selon l'éditeur
Spyware grand public Rare Plus courant

Bottom line : macOS est mieux par défaut, mais les deux nécessitent du hardening manuel pour atteindre le niveau "vraiment privé".

🛡️ Le combo gagnant à mettre en place ce week-end (Mac)

Pour faire vite : Bitwarden (mots de passe) + Mullvad ou NordVPN (VPN) + FileVault activé + pare-feu ON. Tu peux être en mode "Mac vraiment privé" en moins de 2 heures.
